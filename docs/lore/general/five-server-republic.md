# Пятая Эррландская Всесерверная республика

**Пятая Эррландская Всесерверная республика (Пятая Всесерверная, Вторая Республика Всех Эррландцев)** - политическая концепция [национал-сентябристов](/Sentyabrizm-08-05), предполагающая создание Всесерверного [Эррландского государства](/EHrrlandskaya-Narodnaya-Sovetskaya-Respublika-05-27) в условиях [неэрийского сервера](/YAvlenie-Bezdomnoj-EHrrlandii-08-05) (вне New Erы) при помощи прямого подчинения (эррландоинтеграции) или вассализации через создание Сентябристских республик (проект Астральской Сентябристской Республики, Винериумской Сентябристской Республики и т.п).

Считается основной внешнеполитической доктриной [КВО ЭНСР](/OVR-EHNSR-06-30).

Реализована на New Era
