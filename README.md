# Большая Энциклопедия NewEra и Эррландии

Добро пожаловать в репозиторий энциклопедии Эррландии! Здесь хранятся все статьи, изображения и настройки сайта, собранные с помощью [MkDocs](https://www.mkdocs.org/) и темы [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

## Для разработчиков (локальный запуск)

Если вы хотите развернуть сайт на своём ПК для тестирования или правок:

1. Установите Python 3.8+ и `pip`.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите локальный сервер:
   ```bash
   mkdocs serve
   ```
4. Откройте в браузере `http://127.0.0.1:8000`.
